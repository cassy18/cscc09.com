Cascading Buttons

These buttons all match .btn, but the more specific rules for .btn.btn-primary and .btn.btn-secondary override some of the base styles. 

.btn {
  background-color: white;
  color: black;
}

.btn.btn-primary {
  background-color: blue;
  color: white;
}

.btn.btn-secondary {
  background-color: lightgray;
}
